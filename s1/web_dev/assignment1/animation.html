Animations I Used

I have used several elements in my webpages that have given an animation:

  1. In my navigation bar, I have made the page link that the user are currently in fade from the standard dark-grey colour to a light blue color to indicate what page the user is on. I have used this in all my pages but it can be found in the Home, Interests, Timetable and Animation pages.

  2. When the user hovers their cursor over a link in the navigation bar, the underline to the link fades in. The underline also fades out when the cursor is removed from the link. This can also be found on all pages, which includes the Home, Interests, Timetable and Animation pages.

  3. In the Home page, I set my image to transpose from the left border of the screen before resting at it's desired place in the information box.

  4. In the Interests page, I set my image to transpose from the right border to it's desired resting place.

  5. In the Timetable page, I made my image grow in size into it's place.

  6. And in the Animation (this) page, I made my image fade into the page from 0% opacity to 100%
